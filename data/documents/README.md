# Sample Documents

This directory contains documents to be ingested into the Second Brain knowledge base.

## Supported Formats
- Markdown (.md)
- Plain text (.txt)
- PDF (requires additional dependencies)

## Adding Documents

1. Place your documents in this directory
2. Run the ingestion script or use the `add` command in the interactive CLI

## Example

```
data/documents/
├── notes.md
├── research_paper.txt
└── project_docs/
    └── overview.md
```
